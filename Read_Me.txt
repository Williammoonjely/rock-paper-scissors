

To Run this Code follow the steps given below:
1. open terminal in the downloaded folder
2. create the virtual environment using command : virtualenv venv
3. activate the virtual environment using command : source venv/bin/activate
4. install the support packages using command : pip install -r support_file.txt
5. run python code using command : python rock_paper_scissors.python
6. copy paste the address link in default browser address bar
7. enjoy :)
